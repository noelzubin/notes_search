# Readme 
An indexer and TUI searcher for my notes.

Written using: 
  bleve ( text indexing )
  bubbletea ( TUI )


Sample config
``` yaml
root_path: /Users/username/Dropbox/wiki
editor: hx
extensions: 
  - .md
  - .rs
```
